API Quick Start Guide

Steps to run the API properly:

  • Create a database (MySQL) named fluxstore (optional).
  • Create a .env file and fill it with the following values (copy):

    • PORT=4000
    JWT_SECRET=kajdhawk812831ohadkjahd@@0102`\098*&
    DATABASE_URL="mysql://username:password@host:port/database_name"

  • Replace username, password, host, and port according to your database.

  • Then run Prisma with the following code in the terminal:

    • npx prisma generate
    npx prisma db push


  • If you want to use Prisma Studio to view the database flow, use:

    • npx prisma studio


  • finally run the API with the following command:

    • pnpm run dev

    or if you want to use npm

    npm run dev

    Next, there is documentation for the API routing list, but you must log in with a user that is already registered in the database to access
