Sort Product by Category

Endpoint: GET /sortProductCategory

Parameters:

  • id (optional): ID kategori.
  • nameCategory (optional): Nama kategori.

Deskripsi: Mendapatkan daftar produk yang termasuk dalam kategori tertentu, jika tanpa parameter maka akan mengambil semua data.

Sort Sub Categories

Endpoint: GET /sortSubCategories

Parameters:

  • id (optional): ID sub-kategori.
  • nameSubCategory (optional): Nama sub-kategori.
  • categoryId (optional): ID kategori.

Deskripsi: Mendapatkan daftar sub-kategori produk, jika tanpa parameter maka akan mengambil semua data.

Sort Products

Endpoint: GET /sortProducts

exp : http://localhost:3000/sortProducts?name=Dress

Parameters:

  • name (optional): Nama produk.
  • gender (optional): Gender produk(male or female).
  • minDiscount dan maxDiscount (optional): Rentang diskon produk.
  • minPrice dan maxPrice (optional): Rentang harga produk.
  • id (optional): ID produk.

Deskripsi: Mendapatkan daftar produk berdasarkan berbagai kriteria seperti nama, gender, diskon, dan harga, jika tanpa parameter maka akan mengambil semua data product.

Get Product by ID

Endpoint: GET /sortProducts/:id

Deskripsi: Mendapatkan detail produk berdasarkan ID.

Show Single Product

Method: GET

required Fields param: id

Endpoint: /product/show/:id

exp param : http://localhost:3000/product/show/123131kaljdakadj

Delete Product

Method: DELETE

required Fields param: id

Endpoint: /product/delete/:id

exp param : http://localhost:3000/product/delete/123131kaljdakadj

Delete Detail Product

Method: DELETE

required Fields param: id

Endpoint: /product/deleteDetail/:id

exp param : http://localhost:3000/product/deleteDetail/123131kaljdakadj

Update Product

Method: PUT

required Fields param: id

required Fields body: nameProduct, priceProduct, descProduct(optional), starProduct(optional), categoryProduct, gender(optional), rekomendedProduct(only true, default false)

Endpoint: /product/update/:id

exp param : http://localhost:3000/product/updateDetail/123131kaljdakadj

Example body JSON:

      
        {
          "nameProduct": "iphone 12",
          "priceProduct": "25000000",
          "starProduct": "4",
          "categoryProduct": "electronic",
          "gender": "uni",
          "rekomendedProduct": true
        }
